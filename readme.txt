简介：该通讯协议可以让STM32F407IGHX（即RoboMasterC板）与安装有ROS2的Ubuntu20.04/或ROS2进行串口通信。
该协议是利用USB-CDC，创建了一个虚拟串口(名称为/dev/ttyACM0)，从而让下位机可以通过micro-USB与电脑进行连接，并进行串口通信
该文件夹包含下位机STM32端的关键代码，而且基本上已经写好了足够的注释，我也会在相应位置写出一些注意。
